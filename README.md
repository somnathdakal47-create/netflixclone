# Netflix Clone

A responsive Netflix-inspired web application built with HTML and CSS. This project replicates the core UI and styling of the Netflix platform.

## 🎬 Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI** - Clean and intuitive user interface inspired by Netflix
- **CSS Styling** - Professional styling with modern CSS techniques
- **Static Content** - HTML structure with beautifully styled components

## 🛠️ Tech Stack

- **HTML5** - Semantic markup structure (32.2%)
- **CSS3** - Responsive styling and animations (67.8%)

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional installation required

### Installation

1. Clone the repository:
```bash
git clone https://github.com/somnathdakal47-create/netflixclone.git
```

2. Navigate to the project directory:
```bash
cd netflixclone
```

3. Open in your browser:
   - Simply open the `index.html` file in your web browser
   - Or use a local server:
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Node.js (http-server)
     npx http-server
     ```

4. Visit `http://localhost:8000` in your browser
